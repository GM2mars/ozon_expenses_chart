# График трат пользователя

Это приложение позволяет отслеживать и визуализировать расходы пользователя в виде графика.

## Запуск

Для установки зависимостей и запуска приложения используйте:

### С использованием Bun:
```
bun i
bun dev
```

### С использованием NPM:
```
npm i
npm run dev
```

После запуска приложение будет доступно в браузере по адресу http://localhost:5173
